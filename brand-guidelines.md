## Beacon House Hero Design System

### Color Palette

```css
--beacon-navy: #203A6B;
--deep-ink: #14213D;
--warm-amber: #F8AF48;
--soft-blue-mist: #F4F7FB;
--warm-ivory: #FFFBF5;
--slate-text: #536174;
--border-blue-gray: #DDE5EE;
--white: #FFFFFF;
```

### Color Usage

Use `#FFFBF5` and `#F4F7FB` as the main hero backgrounds. The page should feel light, calm, premium, and breathable.

Use `#14213D` for the main headline and high-importance text.

Use `#203A6B` for brand elements, stat numbers, nav/header text, and selected emphasis.

Use `#536174` for body copy, stat labels, secondary descriptions, and muted text.

Use `#DDE5EE` for card borders, dividers, input borders, and subtle layout structure.

Use `#F8AF48` only for the primary CTA and very small accents. Do not use amber as a large background block. It should feel restrained and premium, not loud.

### Font Pairing

Use an elegant editorial serif for headlines, preferably `Lora`.

Use a clean modern sans-serif for everything else, preferably `Plus Jakarta Sans`.

```css
--font-heading: "Lora", Georgia, serif;
--font-body: "Plus Jakarta Sans", system-ui, sans-serif;
```

### Font Usage

Hero headline: `Lora`, weight `500–600`, color `#14213D`, large size, generous line-height.

Body copy: `Plus Jakarta Sans`, weight `400`, color `#536174`, line-height `1.6–1.75`.

Stats numbers: `Lora` or `Plus Jakarta Sans`, weight `500–600`, color `#203A6B`.

Stats labels: `Plus Jakarta Sans`, weight `400–500`, color `#536174`.

CTA text: `Plus Jakarta Sans`, weight `600`, color `#14213D`.

Buttons: background `#F8AF48`, text `#14213D`, rounded rectangle, no pill shape, no heavy glow.

### Overall Rule

The visual tone should feel like a trusted admissions advisory firm: calm, senior-led, institutional, premium, and precise. Avoid startup gradients, playful fonts, handwritten accents, loud yellow sections, heavy shadows, and anything that feels like a rushed marketing funnel.
